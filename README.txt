# OpenPDF
OpenPDF es un proyecto open source que tiene como finalidad la manipulacion y conversion de archivos PDF sin costo alguno.

Requisitos para su ejecucion (Localmente):
- XAMPP | https://www.apachefriends.org/index.html
- PHPOffice => PHPWord | PhpSpreadsheet | https://github.com/PHPOffice
- mpdf | https://mpdf.github.io/
- PDFMerger | https://github.com/clegginabox/pdf-merger

Funciones
- Convertir Word a PDF
- Convertir Excel a PDF
- Convertir imagenes PNG, JPG y JPEG a PDF
- Modificacion de un archivo PDF (Contenido)
- Unificar PDF
- Establecer contrase√±a al PDF
- Editar/Cortar PDF
